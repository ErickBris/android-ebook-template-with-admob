Which Eclipse version is needed? - top

Latest Eclipse version is recommended, which can be downloaded from here:
http://developer.android.com/sdk/index.html

How to open the project in Eclipse? - top

  1. Open Eclipse > Import > Android > "Existing Android Code into Worksplace" > Next
  2. Root Directory > Click Browse button & browse the project folder “Android eBook Template” > Select all 2 projects 'Android eBook Template' and 'google-play-services_lib'
  3. After sucessfully importing the app, Right Click 'Android eBook Template' project from Package Explorer > Properties > Java Build Path > Order and Export > Check on 'Android Private Libraries' > OK
  4. Then go to 'Project' Menu > Clean.. > Clean All Projects > OK
  5. Wait for few minutes until all errors are gone. Ignore any warning appears in 'Problems' tab.

How to change the package name? - top

You can change it easily from Eclipse. Here are the steps:

  • Right Cilck 'Android eBook Template' project from Package Explorer > Application Tools > Rename Application Package > Finish.

How to change app name? - top

  1. Open Eclipse > Android eBook Template > res > values > strings.xml
  2. Select 'strings.xml' tab next to 'Resources' tab
  3. Enter your app name inside "app_name" string tag:
    <string name="app_name">Android eBook Template</string>
This will change both title inside the app & app icon name.

How to add new part/story/chapter? - top

You have to walk through 2 steps to add a new story
  1. Create new part or category
  2. Add HTML file
  3. Add link of file to structure file

 

To create new part or category you need following 3 steps.

A. Create new part or category:
  1. Go to: Eclipse > Android eBook Template > assets
  2. Create new folder for part/category. In this template, there are already two parts: part01 and part02
 
B. Add HTML file
  1. Go to: Eclipse > Android eBook Template > assets
  2. Select part or category you have created
  3. Create HTML file contains content of this part
 
C. Add link of file to structure file
  1. Go to: Eclipse > Android eBook Template > assets
  2. Select 'strings.xml' tab next to 'Resources' tab
  3. Enter new line for your part with syntax:

    physic_folder_category_name@Academic Category Name@Physical Chapter file's name@Academic Chapter's name

How to change AdMob publisher id? - top

  1. Open Eclipse > Android eBook Template > res > values > ads.xml
  2. Replace old admob IDs by yours Admob ID in banner_ad_unit_id(Banner Ads) and full_ad_unit_id (Intestitial Ads)

How to change 'About' section? - top

  1. Go to: Eclipse > Android eBook Template > res > values > contact.html
  2. Change all information by yours, note that you have to change the developer_name to reference to right rating page

Graphics - top

You will find all buttons & station image into /res/drawable/ and /res/drawable-hdpi/ folder. Just create your own graphics and replace those files. Create seperate graphics of different device screen sizes & place them into other drawble folders (drawable-mdpi, drawable-ldpi & drawable-xhdpi) with same file name found at drawable-mdpi folder. Check here for more information: Devices and Displays & Supporting Multiple Screens.

For example, if you want to change screen image (may be book cover image), you should replace ic_web_launcher.png in res/drawable folder by your image
